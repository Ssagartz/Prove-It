While the below is not actually an axiom of the system, it can take the place of defining Booleans as an enumerated set and defining what an enumerated set is for a set of basic logic axioms that doesn't define any other set that the Booleans.

In [3]:
from proveit._common_ import A
from proveit.logic import Booleans, TRUE, FALSE, Or, Forall, Equals, InSet
Forall(A, Equals(InSet(A, Booleans), Or(Equals(A, TRUE), Equals(A, FALSE))))
